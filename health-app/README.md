# 🏋️ Gym Guide for Women

여성분들이 헬스장에서 혼자서도 운동기구를 쉽게 사용할 수 있도록 돕는 웹사이트입니다.

## 🚀 시작하기

이 프로젝트는 별도의 설치가 필요 없는 정적 웹사이트입니다.

1. `health-app` 폴더의 `index.html` 파일을 더블 클릭하여 실행하세요.
2. 웹 브라우저에서 바로 사용할 수 있습니다.

## 📱 주요 기능

- **기구 검색**: 이름이나 운동 부위로 검색
- **부위별 필터**: 상체, 하체, 유산소 등 카테고리별 보기
- **상세 가이드**: 초보자 눈높이에 맞춘 단계별 사용법 및 꿀팁
- **권장 중량**: 여성 초보자 기준 권장 무게 제시

## 📂 구조

- `index.html`: 모든 기능이 포함된 메인 파일 (HTML/CSS/JS)
