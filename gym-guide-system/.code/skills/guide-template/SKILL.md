# 운동기구 사용법 가이드 작성 템플릿

이 템플릿은 모든 운동기구 사용법 가이드를 일관되고 이해하기 쉽게 작성하기 위한 표준 구조입니다.

---

## 📐 기본 구조

```
# 🏋️ [기구명] 사용 가이드

## 📍 이 기구는 이럴 때 사용해요
[기구의 목적과 효과]

## ✅ 사용 전 체크
[체크리스트]

## 📝 단계별 사용법
[3-5단계]

## ⚠️ 이것만은 꼭 기억하세요!
[주의사항]

## 💪 추천 운동량
[세트/횟수]

## 🎥 더 알아보기
[참고 자료]
```

---

## 🎨 섹션별 작성 가이드

### 1️⃣ 헤더 섹션

- 이모지는 기구 종류에 맞춰 선택
- 기구명은 한글명 우선, 영문명은 괄호 안에

**예시**:
- `# 🏋️ 레그프레스 (Leg Press) 사용 가이드`
- `# 💪 랫풀다운 (Lat Pull Down) 사용 가이드`

---

### 2️⃣ 기구 소개 섹션

```
## 📍 이 기구는 이럴 때 사용해요

**타겟 부위**: [주요 운동 부위]
**난이도**: ★ / ★★ / ★★★
**주요 효과**:
- [효과 1]
- [효과 2]
- [효과 3]

**이런 분께 추천해요**:
- [추천 대상 1]
- [추천 대상 2]
```

**작성 팁**:
- equipment-database에서 기본 정보 가져오기
- 효과는 구체적으로 (예: "탄탄한 허벅지" 대신 "허벅지 앞쪽 선명한 라인")
- 초보자 관점에서 이해하기 쉽게

---

### 3️⃣ 사용 전 체크리스트

```
## ✅ 사용 전 체크

시작하기 전에 이것들을 확인하세요:

- [ ] **중량 설정**: 초보자는 [N]kg부터 시작 (바를 움직여서 핀 꽂기)
- [ ] **시트/패드 높이**: [조절 방법 상세 설명]
- [ ] **안전장치 확인**: [해당되는 경우]
- [ ] **발판/손잡이 위치**: [정확한 위치]

💡 **처음이라면**: 일단 가장 가벼운 무게로 동작을 익혀보세요!
```

**작성 팁**:
- 체크박스 사용으로 실용성 높이기
- 구체적인 숫자 제시 (초보자 권장 중량)
- 조절 방법도 상세히 ("빨간 레버를 위로 당기면...")

---

### 4️⃣ 단계별 사용법 (핵심!)

```
## 📝 단계별 사용법

### 1단계: 시작 자세 🪑

**앉는 위치**:
- [상세 설명]
- [그림으로 상상할 수 있도록 묘사]

**발 위치**:
- [발을 어디에 두는지]
- [발 너비, 각도]

**손/등/허리 자세**:
- [손잡이 잡는 법]
- [등은 시트에 붙이기 등]

💡 **포인트**: [이 단계에서 가장 중요한 팁]

---

### 2단계: 동작 실행 💪

**움직이는 방법**:
- [구체적인 동작 설명]
- [범위: 어디까지 밀거나 당기는지]

**호흡법**:
- 🌬️ **들이쉬기**: [언제]
- 💨 **내쉬기**: [언제]

**속도**:
- [천천히 vs 빠르게]
- [예: "2초에 걸쳐 밀고, 2초에 걸쳐 돌아오기"]

💡 **포인트**: [이 단계에서 가장 중요한 팁]

---

### 3단계: 마무리 🔄

**반복**:
- [몇 회 반복할지]
- [세트 사이 휴식]

**중량 내리는 방법**:
- [안전하게 끝내는 법]

**마무리 후**:
- [가벼운 스트레칭 추천]

💡 **포인트**: [마무리 팁]
```

**작성 팁**:
- 각 단계마다 이모지로 구분
- 호흡법은 필수!
- "포인트"로 핵심 강조
- 초보자가 상상할 수 있도록 구체적으로

### 5️⃣ 주의사항 섹션

```
## ⚠️ 이것만은 꼭 기억하세요!

### ❌ 흔한 실수 TOP 3

1. **[실수 1 - 구체적으로]**
   - 왜 안 좋나요? [설명]
   - 어떻게 해야 하나요? [올바른 방법]

2. **[실수 2]**
   - 왜 안 좋나요? [설명]
   - 어떻게 해야 하나요? [올바른 방법]

3. **[실수 3]**
   - 왜 안 좋나요? [설명]
   - 어떻게 해야 하나요? [올바른 방법]

### ✅ 안전 수칙

- 🚫 **이런 경우 하지 마세요**: [부상, 통증 등]
- ⚠️ **주의할 점**: [특정 신체 부위 보호]
- 💊 **통증 느껴지면**: 즉시 중단하고 트레이너 상담

### 💡 효과를 높이는 팁

- [팁 1]
- [팁 2]
- [팁 3]
```

**작성 팁**:
- "흔한 실수"는 구체적인 행동으로 (예: "등을 시트에서 떼고 하기")
- 왜 안 좋은지 + 어떻게 해야 하는지 세트로
- 안전 수칙은 반드시 포함

### 6️⃣ 운동량 가이드

```
## 💪 추천 운동량

### 초보자 (첫 1-2주)
- **세트**: 3세트
- **횟수**: 10-12회
- **중량**: [N]kg
- **휴식**: 세트 사이 60-90초

**목표**: 정확한 자세 익히기!

### 중급자 (3주차 이후)
- **세트**: 3-4세트
- **횟수**: 12-15회
- **중량**: [N]kg (조금씩 올리기)
- **휴식**: 세트 사이 45-60초

**목표**: 근지구력 향상!

---

💬 **Tip**:
처음에는 가벼운 무게로 자세를 완벽하게 익히는 게 가장 중요해요!
무게는 천천히 올려도 늦지 않아요 😊
```

**작성 팁**:
- 초보자/중급자 명확히 구분
- 구체적인 숫자 제시
- 목표를 명시하여 동기부여

## ✍️ 작성 원칙 (중요!)

### 1. 언어 스타일
- ✅ **쉬운 일상어 사용**: "대퇴사두근" → "허벅지 앞쪽"
- ✅ **친근한 톤**: "~해요", "~하세요"
- ✅ **격려하는 말투**: "할 수 있어요!", "걱정 마세요"
- ❌ **전문 용어 최소화**: 필요하면 괄호로 설명

### 2. 구조
- ✅ **이모지 적극 활용**: 가독성 ↑
- ✅ **체크박스, 번호**: 따라하기 쉽게
- ✅ **단계는 3-5개**: 너무 많으면 복잡
- ❌ **긴 문단 지양**: 짧고 명확하게

### 3. 안전 우선
- ✅ **부상 방지 정보 강조**
- ✅ **통증 시 중단 명시**
- ✅ **초보자 권장 중량 제시**
- ❌ **과한 목표 제시 금지**: 천천히, 안전하게

### 4. 공감과 격려
- ✅ **초보자의 두려움 공감**
  - "처음엔 다들 어려워요"
  - "천천히 하면 돼요"
- ✅ **자신감 부여**
  - "혼자서도 충분히 할 수 있어요!"
  - "당신은 잘 하고 있어요"
- ❌ **비교하거나 압박하지 않기**

### 5. 시각화
- ✅ **상상할 수 있게 묘사**
  - 나쁜 예: "정확한 자세로"
  - 좋은 예: "등을 시트에 완전히 붙이고, 발은 어깨 너비로"
- ✅ **구체적인 기준 제시**
  - "90도 각도까지 내리기"
  - "발판 중간 부분에 발 위치"
